
Objective
Health chatbot using Google's Gemini API with prompt engineering and safety filters.

Model
Gemini 2.5 Flash via Google's genai library

Features
System prompt for medical assistant persona

Safety disclaimers built into prompts

Concise, to-the-point answers

Exit command to end chat

Code Structure
SYSTEM_PROMPT - Defines bot behavior and safety rules

ask_health_bot() - Sends queries to Gemini API

while loop - Interactive command-line chat
